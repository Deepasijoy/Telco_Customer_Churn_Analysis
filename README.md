# Telco_Customer_Churn_Analysis
Telco Customer Churn Analysis using Machine Learning. Performed EDA, feature engineering, feature importance analysis, and predictive modeling to identify key drivers of customer churn.

## Project Overview
Customer churn is one of the biggest challenges faced by telecom companies. Acquiring new customers is significantly more expensive than retaining existing customers. This project aims to understand customer behavior, identify the factors influencing churn, and build an ML model to predict which customers are likely to leave the company.

## Objective 
Predict customer churn and identify the most influential factors affecting customer retention and attrition.

## Key Insights
  - Longer tenure strongly reduces churn
  - Fiber Optic customers show higher churn rates
  - The longer tenure contracts improve sutomer retention

## Recommendation
- Promote long-term contracts
- Improve onboarding for new customers
- Investigate churn among Fibre Optic users
- Review pricing for high monthly -charged customers
